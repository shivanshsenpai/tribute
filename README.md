# tribute

A personal tribute webpage for my Activa, Yezdi Roadster, and CBR 250R Repsol.

## Run locally

```bash
python -m http.server 8000 --bind 127.0.0.1
```

Then open:

```text
http://127.0.0.1:8000/
```

The page includes original photos, AI-generated tribute art, and a code-built 3D memory garage.
